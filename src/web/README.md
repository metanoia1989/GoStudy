# 使用的工具和技术栈     

## PostgreSQL 
Windows 安装 https://www.postgresql.org/download/windows/

Linux 安装 
```sh
$ sudo apt install postgresql postgresql-contrib
$ sudo su postgres
$ createuser -interactive # 创建 PostgreSQL 账号
$ createdb <Your Account Name>
```
